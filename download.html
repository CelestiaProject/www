Download

Please select the appropriate Celestia package for your computer from the list below.

Mobile Celestia versions are available at celestia.mobi

Windows

The Windows package of Celestia is a self-extracting archive. Download it to your computer and then run it.

  • Celestia 1.6.2 (x86/x64, 36M)
  • Celestia 1.6.1 (x86, 33M)

Linux

If you are running Debian or Ubuntu, see the instructions at the link below.

  • Get Celestia (x86/x64) from Bintray

macOS

The macOS packages for versions 1.6.0 and 1.6.1 are disk image. Download one to your computer, double click it, and follow the instructions in the README. The package for 1.6.2 is a zipped bundle, download it and unpack.

  • Celestia 1.6.2 (41M)
  • Celestia 1.6.1 (37M)
