Download

Please select the appropriate Celestia package for your computer from the list below.

Mobile Celestia versions are available at celestia.mobi

Windows

The Windows package of Celestia is a self-extracting archive. Download it to your computer and then run it.

  • Celestia 1.7.0 (x86/x64)
  • Celestia 1.6.4 (x86/x64, 36M)
  • Celestia 1.6.3 (x86/x64, 36M)

macOS

The macOS packages are zipped bundles, choose your version, then download it and unpack.

  • Celestia 1.6.4 (41M)
  • Celestia 1.6.3 (41M)

GNU/Linux

See the installation instructions for your distro at the below pages.

  • Debian (x86/x64/arm64)
  • openSUSE (x86/x64/arm64)
  • Ubuntu (x86/x64/arm64)
  • AppImage (x64)

FreeBSD

The Freebsd package of Celestia is available through FreeBSD's repositories via pkg or as a port. See the installation instructions on the below page.

  • FreeBSD (x86/x64)
