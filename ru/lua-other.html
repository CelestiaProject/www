Lua-инструменты

Сообщество разработало несколько полезных Lua-инструментов для расширения возможностей Celestia. Ниже приведен краткий обзор некоторых из них:

Lua Edu Tools v1.2 by Vincent Giangiulio and Hank Ramsey
LUA Edu Tools была разработана в первую очередь для использования в учебных заведениях (школах, университетских институтах) и направлена на облегчение работы пользователей с программой Celestia. В то же время это был инструмент, который можно использовать для быстрой организации познавательных занятий по астрономии. Со временем это дополнение к Celestia быстро завоевало популярность и среди обычных пользователей, поскольку предоставило программе дополнительные функции, недоступные в официальной версии.
Затем Lua Edu Tools был переведен на несколько языков (включая русский), и мы рекомендуем использовать его всем пользователям Celestia.
Итак, что же такое Lua Edu Tools?

Это дополнение представляет собой графический интерфейс для управления программой Celestia, который состоит из следующих элементов:
- Установка даты/времени, управление временем (ускорение, замедление)
- Каталог Солнечной системы с классификацией объектов по типу (планеты, спутники, астероиды и т. д.)
- Настройка рассеянного света
- Настройка света галактик
- Удобные кнопки навигации
- Настройка параметров отображения
- Настройка FOV
- Пояс астероидов
- Панель расширения (дополнительные модули)
- Декоративная панель с текстовой информацией
- Декоративная панель с изображением
- Режим планетария/навигации
- кнопки навигации и доп. элементы управления движением
- функция масштабирования (предназначена для образовательных целей и отображения солнечной системы в другом масштабе)
- Различные маркеры
- Компас
И это не предел. LUA Edu Tools позволяет расширять и добавлять новые функции и команды. Преподавателям предоставлены большие возможности для создания "живых" презентаций по астрономии (например с помощью сценариев) и захватывающих путешествий в симуляторе Celestia. Любителям научной фантастики и видеоигр можно включить показ кокпитов, имитацию приборной панели NASA Space Shuttle (или других) и вид из иллюминатора. Это создает особую атмосферу пребывания на борту космического корабля и дает пользователю незабываемый опыт путешествий по вселенной Celestia.

Slideshow by Jogad

SLIDESHOW позволяет создавать сложные презентации в Celestia. Вы можете разнообразить экран Celestia текстом и изображениями, которые можно по желанию форматировать. Слайд-шоу представляет собой набор слайдов, каждый из которых может хранить позицию в Celestia (Cel:// URL) вместе с изображениями и текстом. Кроме того, в каждый слайд можно также вставлять разные файлы, например мультимедийные файлы (аудио или видео), текстовые или PDF-документы, интернет-ссылки или даже скрипты с командами Celestia. Слайд-шоу теперь снабжается шрифтами для изменения размера и формы текста и улучшения внешнего вида слайдов. Вы можете создать столько слайд-шоу, сколько пожелаете. Каждое слайд-шоу может содержать столько слайдов, сколько необходимо.
SLIDESHOW может легко перемещаться между слайдами, а также автоматически прокручиваться с регулируемым интервалом времени. Все это можно сделать без каких-либо навыков программирования или написания скриптов.
