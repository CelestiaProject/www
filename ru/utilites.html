Утилиты

Сообщество разработало несколько полезных инструментов для Celestia. Вот краткий обзор некоторых из них:

CmodView
CmodView - это программа, которая может загружать и отображать собственный формат .cmod Celestia, используемый для 3D-моделей. CmodView показывает модель, как она будет выглядеть в программе, и предлагает полезные функции для редактирования некоторых ее свойств, которые не могут быть сделаны другими программами. При сохранении файла .cmod он преобразуется в более компактный двоичный формат cmod, что повышает производительность в Celestia. Кроме того, cmodview также может напрямую загрузить более распространенный формат .obj и преобразовать его в файл .cmod.

Инструменты для карт нормалей и виртуальных текстур (VT)

Набор инструментов командной строки, которые помогут создавать высококачественные карты нормалей и виртуальные текстуры для использования в Celestia, доступные для Windows, Linux и MAC. NmTools сосредоточен на преобразовании топографических данных в карты нормалей, чтении двоичных файлов и выводе текстуры, которую можно использовать в Celestia для отображения динамического затенения рельефа. F-TexTools создает виртуальные текстуры из существующих файлов текстур. Виртуальные текстуры состоят из мозаичных изображений, которые можно подгружаются в зависимости от увеличения для отображения поверхности планеты в очень высоком разрешении. Отдельные программы можно объединить в одну команду, чтобы оптимизировать процесс, не создавая бесполезных полуобработанных файлов. Инструменты для текстур дополнены внешними программами для распаковки и преобразования текстур.

DDS конвертер

Программа для преобразования текстур из формата .jpg или .png в формат .dds с использованием NVIDIA Texture Tools. Файлы DDS загружаются в Celestia быстрее, что повышает производительность. Утилита может загружать целые папки текстур и быстро конвертировать их.
