Celestia installation on openSUSE

Celestia 1.6 on openSUSE Leap/Tumbleweed:

								
sudo zypper addrepo https://download.opensuse.org/repositories/home:munix9:celestia:1.6/${VERSION}/home:munix9:celestia:1.6.repo
sudo zypper refresh
sudo zypper install celestia

Where VERSION is 15.6, 16.0, 16.1 or openSUSE_Tumbleweed.

See also the download package site on OBS for celestia.
