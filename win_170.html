Celestia 1.7.0 installation on Windows

Celestia 1.7.0 is a rolling release, meaning that it will update relativly frequently, so check back constantly for updates. Celestia 1.7.0 on Windows also comes in multiple versions that can be downloaded below.

  • Celestia 1.7.0 for x64 based Windows Systems (345 MB)
    This version will run on any computer with an amd64 type CPU.
  • Celestia 1.7.0 for x86 based Windows Systems (332 MB)
    This version will run on any computer with either an i686 or amd64 type CPU.
  • Source Code for 1.7.0 (ZIP Archive)

  • Source Code for 1.7.0 (TAR.GZ Archive)
    If you want to compile Celestia from source, you can download the source code directly.
